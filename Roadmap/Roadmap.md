# Invinsible Tech Website Design Roadmap

## Website Goal

Build a premium, futuristic, dark-themed company website inspired by the attached Factory-style design.

The website should make Invinsible Tech look:

* Modern
* AI-first
* Premium and enterprise-ready
* Trustworthy for business clients
* Strong in automation, AI, software, and digital transformation

The design direction should feel similar to high-end SaaS, AI, and technology companies.

## Design Style Direction

### Overall Visual Style

* Dark background with subtle grid textures
* White typography with orange accent color
* Thin borders, rounded cards, glowing elements
* Smooth animations and micro-interactions
* Large bold headlines
* Minimalist but futuristic UI
* Sections separated by large spacing
* Floating UI cards and dashboard mockups
* Subtle 3D objects, lines, graphs, particles, and animated dots
* Use motion effects while scrolling
* Use blur, glassmorphism, and gradient lighting

### Fonts

* Primary Font: Inter
* Secondary Font: Space Grotesk or Sora

### Colors

* Background: Black / Dark Gray
* Primary Text: White
* Secondary Text: Gray
* Accent Color: Orange
* Optional Supporting Accent: Neon Blue
